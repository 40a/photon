photon
======

A data driven tool to deploy fusion.

Usage
-----

Execute ansible for the given az.

.. code-block:: bash

	$ photon deploy --az <az_name>

Testing
-------

Requirements:

* Tox

Execute unit tests:

.. code-block:: bash

	$ tox
